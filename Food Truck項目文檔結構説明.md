# 項目文檔結構説明

| 目錄       | 説明                                   |
| ---------- | -------------------------------------- |
| lib        | 根目錄                                 |
| ./pages    | 存放各個頁面的Widget                   |
| ./constant | 存放一些常量類型文件，如項目的常用樣式 |
|./model|存放數據庫文件|
|./main.dart|函數入口文件|
|./assets/|資源|
|./assets/images|圖片|
|./router|路由管理|
|./util|公用的方法/控件庫|
|./state|狀態管理器|
|./widgets|公用的控件|
